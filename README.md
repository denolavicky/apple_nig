# Apple iPhone Page — Design Project

## Overview
A pixel-perfect redesign of the [apple.com/ng](https://apple.com/ng) landing page and the iPhone product page, built with semantic HTML and vanilla CSS — no JavaScript, no frameworks.

## Pages
- **Landing Page** — apple.com/ng homepage
- **iPhone Page** — iPhone product page

## Tech Stack
- HTML5
- CSS3 (Vanilla — no libraries or frameworks)
- No JavaScript


## How to Run
1. Clone the repo
2. Open `index.html` in your browser
3. No build steps or dependencies required
